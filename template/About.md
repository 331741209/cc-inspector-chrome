## 怎么切换node版本
`mac`
> 安装`n`扩展包: sudo npm i n -g    
![](../docs/about/7f59e923.png)

> 安装指定版本node: sudo n 8.4.0    
![](../docs/about/6db62665.png)

> 切换版本: sudo n    
![](../docs/about/bc372062.png)

> 验证node版本: node -v    
![](../docs/about/098511ee.png)
